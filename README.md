# 南雲式WBSツール (Nagumo-style WBS Tool)

Vanilla JS + Tailwind CSS で構築された、超軽量かつ高速な完全自作ガントチャート・プロジェクト管理ツールです。

---

## 🚀 導入・起動方法 (3つのバリエーション)

お使いの環境に合わせて、以下の3つの方法から選んで起動できます。

### 📦 パターンA：【一番簡単】ビルド済みZIPから起動（Python不要）
ノンプログラマーの方や、とりあえず手軽に試してみたい方に最適な方法です。
1. 本リポジトリの右側にある **[Releases](https://github.com/Nagumo-Studio/nagumo-gantt/releases)** ページにアクセスします。
2. 最新バージョンの `NagumoGantt_v*.zip` をダウンロードします。
3. ZIPファイルを任意の場所に解凍します。
4. フォルダ内にある **`NagumoGantt.exe`** をダブルクリックして起動します。
   - 起動すると自動的にブラウザが立ち上がり、サンプルプロジェクトが入ったWBSツールが表示されます。

### ⚡ パターンB：Windowsバッチファイルから起動（Python環境あり）
リポジトリをクローンして簡単に動かしたいWindowsユーザー向けの方法です。
1. 本リポジトリをクローンまたは「Download ZIP」でダウンロードします。
2. 解凍したフォルダ内にある **`WBSツール起動.bat`** をダブルクリックします。
   - 必要な依存ライブラリが自動でインストールされ、ツールが起動します。

### 🛠️ パターンC：マニュアル手動起動（エンジニア向け・他OS対応）
コマンドラインから手動で環境を構築して起動する方法です。
1. 本リポジトリをクローンします。
2. 以下のコマンドで依存ライブラリをインストールします。
   ```bash
   pip install -r requirements.txt
   ```
3. 以下のコマンドでアプリケーションを起動します。
   ```bash
   python app.py
   ```
4. ブラウザから `http://127.0.0.1:5000` にアクセスしてください。

## 🔒 セキュリティとプライバシー（データの保管場所）

本ツールはオフライン動作およびユーザーのデータプライバシー保護を最優先に設計されています。

### 📂 プロジェクトデータ・スケジュールの保存先
- **保存先**: すべてローカルマシンのリポジトリ直下（EXE版の場合はEXEと同階層）にある **`data/` フォルダ内のCSVファイル** に保存されます。
  - プロジェクト別データ： `data/projects/{プロジェクト名}/` 内の各種マスタ＆タスクCSV
  - ツール共通データ： `data/common/m_llm_pricing.csv` （LLMモデルの単価定義＆デフォルトドル円為替レート定義マスタ）
- プロジェクトデータが外部のサーバーやクラウドにアップロードされることは一切ありません。完全にローカル環境で完結します。

### 🤖 AIアシスタント導入ガイドとAPIキーの安全性
本ツールのAIアシスタント機能を利用するためには、各プロバイダーからAPIキーを取得し、ツールの設定画面に入力する必要があります。

#### 🔑 APIキーの取得手順
各プロバイダーのアカウントを作成し、以下のページからAPIキーを発行してください。
- **OpenAI (ChatGPT)**: [OpenAI API Keys](https://platform.openai.com/api-keys) にアクセスし、「Create new secret key」から発行。
- **Anthropic (Claude)**: [Anthropic Console](https://console.anthropic.com/settings/keys) にアクセスし、「Create Key」から発行。
- **Google (Gemini)**: [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセスし、「Create API key」から発行。

#### 🛡️ APIキーの保管・セキュリティについて
- **完全なローカル保管**: 設定したAPIキー、およびAIとの会話履歴は、**各ユーザーのブラウザの「ローカルストレージ (localStorage)」にのみ保存されます。**
- **漏洩ゼロの安全性**: サーバー（FlaskやEXE本体）側、あるいは外部のデータファイル（CSV等）に記録・保管されることは一切ありません。
- そのため、**自分のAPIキーを登録した状態のEXEファイルやプロジェクトフォルダを他の人に配布・共有しても、あなたのAPIキーや会話履歴が他人に漏洩することは100%ありません**。安心してチームメンバーにツールを配布できます。

### 🌐 外部通信が発生するシーン
基本機能は完全にオフラインで動作しますが、以下の機能を使用する時のみ、特定の外部サーバーへの通信が発生します。
1. **AIチャットでアシスタントに相談する時**
   - 設定した各LLMプロバイダー（OpenAI, Anthropic, Google等）のAPIエンドポイントへ、スケジュールデータおよびプロンプトを送信してやり取りを行います。
2. **AI設定画面を開いた時（モデルの動的リスト取得）**
   - 各プロバイダーがサポートしている最新の利用可能モデル一覧を取得するため、外部APIに問い合わせを行います。

## 💡 基本操作マニュアル (Usage)

本ツールは直感的かつ、プロが快適に使える操作感を追求しています。

### 1. タスクの基本操作
- **タスクの移動**: タスクをドラッグ＆ドロップして日付（左右）や行（上下）を自由に移動できます（1日単位でスナップします）。
- **期間の変更**: タスクの左右端をドラッグすることで、直感的に開発期間を伸縮できます。
- **編集・作成**: 
  - タスクのない空白領域で**右クリック** ➜ 「新規タスク作成」を選択すると、その位置のキャラクターに対してタスクを作成できます。
  - すでに存在するタスクを**ダブルクリック**、または**右クリック**から「エディタを開く」で詳細を編集できます。
- **コピー＆ペースト**: タスクを選択して `Ctrl+C` でコピー、空白部分を右クリックして `Ctrl+V` でペーストできます。
- **削除**: タスクを選択して `Delete` キーを押すか、右クリックから削除できます。

### 2. 依存関係（リンク結線）
- **タスクを繋ぐ**: 
  1. 先行タスクの右端にマウスをホバーすると、**赤い丸（コネクタ）**が出現します。
  2. その赤丸をドラッグすると、プレビュー線が伸びます。
  3. 繋げたい後続タスク（または左端のコネクタ）の上でドロップすると、美しいベジェ曲線で結線されます。
- **連動押し出し（ドミノシフト）**: 先行タスクを右（未来）へ移動して後続タスクの開始日を超えると、影響を受けるすべての後続タスクが自動的に押し出されます。
- **リンクの削除**: 依存関係を示す線を**右クリック**すると、専用メニューからその結線のみを削除できます。

### 3. 複数タスクの一括操作（範囲選択）
- `Shift` キーまたは `Ctrl` キーを押しながら、ガントチャートの空白部分をドラッグすると、**半透明のブルーの選択ボックス**が現れます。
- 囲んだタスクが一挙に選択状態（青枠）になり、ドラッグすることで**選択したすべてのタスクを同時に移動**できます。

### 4. 画面表示のカスタマイズ
- **画面のパン移動**: ガントチャートの背景をドラッグ＆ドロップすることで、画面全体を縦横にスクロールできます。
- **日付のズーム**: `マウスホイール` を回転させることで、カーソルのある位置を中心に日付の細かさ（ズームイン・ズームアウト）を変更できます。
- **UI高さの調整**: `Ctrl + マウスホイール` でガントチャートの縦幅（行の高さ）を自由に拡縮できます。
- **イナズマ線**: 画面右上の「イナズマ線」ボタンをONにすると、進捗状況（％）に応じたイナズマ線がオレンジ色の「本日（Today）」線を基準に表示されます。

### 5. AIアシスタント
- 画面右上の「AIチャット」ボタンをクリックするとチャットウィンドウが展開します。
- チャット画面右上の「設定」ボタン（⚙️アイコン）からAPIキー、モデル、温度、およびドル円為替レートを設定してご利用ください。
- チャットでのやり取りごとに、消費したトークン数の内訳（Input / Output）や、モデルの単価に基づいた目安消費金額（ドルおよび指定レートで換算した日本円）がメッセージ下部にリアルタイムで表示され、お財布に優しい開発を支援します。
- 過去のセッション履歴一覧でも、そのセッションでの累計消費トークン数やコストの合計が自動算出されて一覧表示されます。
