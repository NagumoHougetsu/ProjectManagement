# 📝 次世代アートWBSツール 開発引継ぎメモ (Handover)

## 📌 現在の開発進捗 (Completed)
- **インフラ・構成**
  - Python/Flaskによるバックエンド構築完了。
  - `data/projects/` 配下での複数プロジェクトフォルダ切替機能実装。
  - データベース不使用。SVN管理を想定したCSVベースの読込・保存API実装。
  - ポータブル版（EXE化）を見据えた絶対パス解決。
  - ブラウザのキャッシュが強力に効く問題に対し、`?v={{ ts }}` を用いた強制キャッシュバスターを導入。

- **フロントエンド (HTML/CSS/JS)**
  - `Vis.js Timeline` による階層型テトリス配置の実現。
  - サイドパネルエディタおよび、右クリックコンテキストメニューによる進捗変更機能。
  - マスタ管理エディタ（別タブ）の実装。
  - **Undo (取り消し) / Redo (やり直し)** 機能の実装（ボタンおよび `Ctrl+Z`, `Ctrl+Shift+Z` 対応）。
  - CTRL + マウスホイールによる**行（バー）の高さ動的調整**機能。

- **UI/UXデザイン調整**
  - Vis.jsの強制サニタイズ（XSSフィルター）を無効化し、カスタムHTMLとCSSを完全に適用。
  - ガントバーの**角丸化**と、担当者座布団の**左上丸形アイコン化**。
  - 休日マスタとの連動によるタイムライン背景の塗り分け。
    - 平日: 白色（Vis.jsデフォルトのストライプ背景を強制上書き）
    - 土曜: 薄い青色
    - 日祝: 薄い赤色
    - 指定休: 薄いグレー
  - **日付列幅の極限縮小対応と、月内週番号（1w, 2w...）の動的切り替え**実装完了。
  - **CTRL+ホイールによるUI全体のズーム（文字サイズ・バー高さ維持）**実装完了。
